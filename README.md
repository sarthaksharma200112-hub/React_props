# 📌 Job Card UI

A simple React-based UI project that displays job cards with company details such as logo, job role, posting date, tags, salary, and location.

## 🚀 Features

* Clean and responsive card layout
* Displays company logo and job information
* Includes tags like Full-time / Part-time
* Shows salary and location details
* Built using reusable React components

## 🛠️ Tech Stack

* React.js
* HTML
* CSS

## 📂 Project Structure

* `components/` → contains card components
* `App.js` → main file rendering cards

## ▶️ How to Run

1. Clone the repository
2. Run `npm install`
3. Run `npm start`
4. Open `http://localhost:3000`
